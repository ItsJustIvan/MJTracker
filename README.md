# 🀄 MJTracker (v0.1)
**Real-time Scoring Orchestrator for Mission: Mahjong!**

## 🚀 The Vision
A high-legibility, mobile-first scoring tool designed for the Mahjong community in Las Vegas. This tracker moves beyond local state to provide a synchronized, multi-device experience for tournament and casual play.

## 🛠 Tech Stack
- **Frontend:** React + Tailwind CSS (Optimized for Senior Legibility)
- **Backend/Auth:** Supabase (PostgreSQL + Realtime)
- **Hosting:** Vercel
- **Infrastructure:** GitHub Actions for CI/CD
